# DynamicPagination
Create a web page with HTML, CSS(Or Bootstrap), Javascript(Or JQuery) andimplement a simple dynamic pagination functionality. The data can be a list of books with threeproperties: bookId, title, and authorName. You can mock the data inside a Javascript object.There should be 20 books and each page should show maximum 6 books/records. Thepagination links should not load a new page and must use Javascript to dynamically modify andupdate the DOM
